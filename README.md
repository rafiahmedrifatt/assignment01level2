//BLOG 01
TypeScript এর একটি শক্তিশালী ফিচার হলো keyof অপারেটর। এটি মূলত কোনো অবজেক্ট টাইপের সবগুলো key বের করে একটি ইউনিয়ন টাইপ তৈরি করে। এতে কোডে সঠিক key ব্যবহার করা নিশ্চিত হয় এবং ভুল key ব্যবহারের সুযোগ কমে যায়।

keyof আসলে করে কী?

যখন আমরা কোনো অবজেক্টের key গুলোকে টাইপ হিসেবে ব্যবহার করতে চাই, তখন keyof খুব কাজে লাগে। এটি অবজেক্ট টাইপে থাকা প্রতিটি key কে নিয়ে একটি যৌথ টাইপ তৈরি করে।

এর ফলে

ভুল key ব্যবহার করলে TypeScript সাথে সাথে ত্রুটি দেখায়

ফাংশন বা অবজেক্টে dynamic key পাস করেও টাইপ সুরক্ষা বজায় থাকে

example:
type User = {
name: string;
age: number;
email: string;
};

type UserKeys = keyof User;
// এখন UserKeys হলো "name" | "age" | "email"

function getValue(obj: User, key: UserKeys) {
return obj[key];
}

//BLOG 02

TypeScript ব্যবহার করলে আমরা কোডকে শুধু শক্তিশালীই করতে পারি না, বরং আরও সাজানো গোছানোভাবে লিখতে পারি। এমনই একটি ফিচার হলো enum। যখন কোনো নির্দিষ্ট মানের সেট বারবার ব্যবহার করতে হয়, তখন enum আমাদের কোডকে পরিষ্কার, অর্থপূর্ণ এবং ভুলবিহীন রাখতে বড় ভূমিকা রাখে।

enum আসলে কী?

enum বা Enumeration হলো এমন একটি কাঠামো, যার মাধ্যমে আমরা সম্পর্কযুক্ত কয়েকটি স্থির মান (constants) একসাথে সংরক্ষণ করতে পারি।
এটি বিশেষ করে কাজে লাগে যখন

একই ধরনের মান বারবার ব্যবহার করতে হয়

অসংখ্য ম্যাজিক নম্বর বা স্ট্রিং ব্যবহারের ঝামেলা কমাতে চাই

কোডকে আরও readable ও maintainable রাখতে চাই

সংক্ষেপে, enum কোডকে বেশি মানুষের মতো করে পড়া এবং বোঝা যায়।

enum Direction {
Up,
Down,
Left,
Right
}

let move = Direction.Up;
console.log(move);
